Boids é um algoritmo desenvolvido por Craig Reynolds em 1986 que simula o movimento coordenado de um grupo de animais, como um bando de pássaros ou um cardume de peixes.

import { Boids } from '@projects/boids';

<Boids client:only="react" />

Esse modelo implementa os mesmos três comportamentos básicos do modelo original:

- **Alinhamento:** Se move para alinhar-se à direção média local.
- **Coesão:** Se move para a posição média local.
- **Repulsão:** Se afasta para evitar colisões.

Além desses comportamentos básicos, também implementa:

- **Clear:** Evita ficar atrás de outros boids.
- **Target Seeking:** Se move na direção de alvos no campo de visão.

A intensidade de cada comportamento pode ser alterada e partir dessas regras simples, pode-se observar o surgimento de comportamentos emergentes, como formação e divisão de bandos.
